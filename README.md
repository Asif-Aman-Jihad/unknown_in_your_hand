# Cosmic Visualizer

An interactive 3D visualization website that allows you to explore cosmic and biological concepts using hand gestures.

## Features

- **5 Interactive Models**: Earth, Black Hole, Human Body, Supernova, Milky Way Galaxy
- **Hand Gesture Control**: Use your webcam to control models with hand movements
- **Responsive Design**: Works on both desktop and mobile devices
- **Educational Content**: Learn interesting facts about each model
- **Multiple Control Methods**: Use hand gestures, mouse, or touch controls

## How to Use

1. **Allow Camera Access**: When prompted, allow camera access for hand gesture controls
2. **Navigate Models**: Click on the navigation items or use the dropdown to switch models
3. **Interact**:
   - **Hand Gestures**: Rotate, zoom, and select using hand movements
   - **Mouse/Touch**: Use mouse drag to rotate, scroll to zoom
4. **Control Panel**: Adjust settings and view information about each model

## Technologies Used

- Three.js for 3D rendering
- TensorFlow.js for hand pose detection
- Vanilla JavaScript, HTML, CSS
- GitHub Pages for hosting

## Local Development

1. Clone the repository
2. Open `index.html` in a modern browser
3. For full functionality, serve via local server (due to camera access requirements)
